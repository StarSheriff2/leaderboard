### Installation

- Clone this repository
```bash
$ git clone git@github.com:alonsofl/leaderboard.git
$ cd leaderboard/
```

- Run project
```bash
$ npm install
$ npm start # This will run webpack dev server
```

- Or build
```bash
$ npm run build # And open dist/index.html
```
